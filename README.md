# Application JavaFX – Gestion de profils & planning (MVC)

Application développée en JavaFX dans le cadre d’un projet universitaire visant à gérer des profils de secouristes et leur planning.

##  Objectif

Mettre en place une application graphique complète en architecture MVC permettant :

- la gestion de profils
- l’affichage d’un planning dynamique
- la navigation entre plusieurs vues
- une séparation claire Modèle / Vue / Contrôleur

## Architecture

Le projet suit le modèle MVC :

- **Model** : gestion des données
- **View** : interfaces JavaFX (FXML)
- **Controller** : gestion des interactions utilisateur

##  Lancer le projet

### Prérequis

- Java 17+
- JavaFX
- IntelliJ recommandé

### Exécution

Ouvrir le dossier `appli` dans IntelliJ et lancer la classe principale.

##  Compétences mises en œuvre

- Java / JavaFX
- Architecture MVC
- Conception d’interfaces graphiques
- Organisation et structuration du code
